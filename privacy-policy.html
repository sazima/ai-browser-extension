Privacy Policy

Extension: Buddy AI  |  Last updated: March 2026

Summary: Buddy AI does not collect, store, or sell any personal data. All data stays on your device or is sent directly from your browser to the AI API provider you configure. We never see your data.

1. What Data Is Involved

2. How We Collect and Use Data

The extension operates entirely client-side. Specifically:

  • API Key and settings – entered by you in the extension settings panel and stored in your browser's local storage. They are never transmitted to any server controlled by us.
  • Page content – when you ask the extension to perform a task, the current page's text and interactive element information is read from the DOM in your browser and sent to the AI API endpoint you configured (e.g., https://api.deepseek.com/v1 or your own endpoint). This transmission happens directly from your browser to that API provider. We do not proxy, log, or intercept this data.
  • User instructions – the task description you type is sent together with page content to the AI API provider as part of the API request.
  • Chat history – kept in memory only for the duration of your session. It is not persisted to disk or any remote server.

3. Data Sharing

We do not share any data with third parties.

  • The extension has no backend server of its own.
  • The only outbound network request the extension makes is the API call to the endpoint URL you provide in settings (default: https://api.deepseek.com/v1). This goes directly from your browser to that provider.
  • If you use a third-party AI provider (OpenAI, DeepSeek, etc.), their own privacy policy governs how they handle the data you send them.

4. Data Storage

  • All settings are stored using the Chrome storage.local API, which keeps data on your local device and is not synced to Google's servers unless you have Chrome Sync enabled for extensions.
  • We do not use cookies, analytics SDKs, or any tracking technology.
  • No data is ever written to an external database by us.

6. Children's Privacy

This extension is not directed at children under 13. We do not knowingly collect any data from children.
