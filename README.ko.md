# AI 브라우저 어시스턴트

AI가 브라우저 작업을 자동화하는 Chrome 확장 프로그램. 원하는 것을 자연어로 말하면 AI가 브라우저를 조작해 완료합니다.

## 주요 기능

- **자연어 제어** — 하고 싶은 것을 말하면 AI가 처리
- **모든 웹사이트 지원** — 사이트별 설정 불필요
- **실시간 단계 표시** — AI가 무엇을 하는지 항상 확인 가능
- **폼 일괄 입력** — 여러 필드를 한 번에 입력
- **요소 시각화** — 모든 인터랙티브 요소에 번호 표시
- **자동 보호 기능** — 로그인 벽·무한 루프·페이지 멈춤 자동 감지
- **직접 API 키 사용** — DeepSeek, OpenAI 및 OpenAI 호환 API 지원
- **다국어 UI** — English、简体中文、繁體中文、日本語、한국어

### 수동 설치 (개발자 모드)

1. **코드 다운로드** — 이 페이지 우측 상단의 녹색 **Code** 버튼 → **Download ZIP** 클릭 후 압축 해제. 또는 Git 사용: `git clone <저장소 URL>`
2. Chrome을 열고 주소창에 **`chrome://extensions`** 입력 후 Enter
3. 우측 상단의 **개발자 모드** 토글을 켭니다
4. **압축 해제된 확장 프로그램 로드** 클릭
5. 압축 해제된 폴더 안의 **`src`** 폴더 선택 (루트 폴더가 아님)
6. 확장 프로그램 아이콘이 Chrome 툴바에 나타납니다 — 클릭하여 사이드 패널 열기

## 초기 설정

1. 확장 프로그램 아이콘을 클릭해 사이드 패널 열기
2. ⚙️ **설정** 클릭
3. API 키 입력 (DeepSeek / OpenAI / 기타 호환 API 키)
4. 선택사항: 커스텀 **API Base URL** 설정 (기본값: `https://api.deepseek.com/v1`)
5. **저장** 클릭

## 사용 예시

아무 웹페이지나 열고 채팅창에 작업을 입력하세요:

```
GitHub의 sazima 사용자 페이지를 열고 저장소 중 하나에 Star 주기
싱가포르 날씨 확인해줘
내일 베이징에서 상하이 가는 가장 저렴한 항공편 찾아줘
이 페이지 회원가입 폼 작성해줘
```

AI가 각 단계를 실시간으로 표시합니다. **■** 를 클릭하면 언제든지 중지할 수 있습니다.

## 권한 안내

| 권한 | 이유 |
|------|------|
| `<all_urls>` | 사용자가 자동화하려는 모든 웹사이트의 콘텐츠를 읽고 조작하기 위해 |
| `activeTab` | 현재 활성 탭에 접근하기 위해 |
| `scripting` | 페이지 요소를 조작하기 위한 스크립트 주입 |
| `sidePanel` | Chrome 사이드 패널에 채팅 UI 표시 |
| `storage` | API 키와 설정을 기기에 로컬 저장 |
| `tabs` | 활성 탭의 ID와 URL 가져오기 |

API 키는 기기에 로컬로 저장되며, 설정한 AI API 엔드포인트 외에는 어디에도 전송되지 않습니다.
