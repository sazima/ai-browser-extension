Buddy AI 是一款强大的 Chrome 扩展，让你用自然语言自动完成任何网页操作——只需告诉它你想做什么，剩下的交给它来搞定。

它能做什么：
只需用日常语言输入一个任务，AI 就会接管你的浏览器，一步一步地完成它——支持在一次指令中串行完成多个操作。无论是搜索信息、填写表单、浏览复杂网站，还是操作 Web 应用，Buddy AI 都能全自动处理。

可以自动化的任务示例：
- "打开 GitHub 用户 sazima 的主页，并 Star 他的一个仓库"
- "帮我查询新加坡的天气"
- "帮我搜索明天北京到上海最便宜的机票"
- "在亚马逊上找最便宜的机械键盘"
- "帮我填写这个注册表单"
- "去 YouTube 找一个 GitHub 教程并点赞，然后查新加坡天气，最后在 GitHub 的 sazima/test 仓库创建一个催更 issue，附上教程链接和当地天气"

核心功能：
- ✅ 自然语言控制——用日常语言描述任务，无需任何编程
- ✅ 多步骤任务串行——一句话完成复杂的连续操作
- ✅ 支持任意网站——无需针对特定网站做任何配置
- ✅ 实时步骤展示——随时看到 AI 正在执行哪一步
- ✅ 智能批量填表——一次指令填写所有字段
- ✅ 自动保护机制——自动识别登录墙、死循环和页面卡死
- ✅ 使用自己的 API Key——兼容 DeepSeek、OpenAI 及任何 OpenAI 格式接口
- ✅ 灵活配置——可自定义 API Base URL 和最大步数
- ✅ 多语言界面——支持 English、简体中文、繁體中文、日本語、한국어

为什么要安装它？
不要再把时间浪费在重复的浏览器操作上。Buddy AI 让复杂的多步骤网页任务变成一句话。就像拥有一个能操控浏览器的私人助手——数据本地处理，隐私完全由你掌控。

隐私说明：
你的 API Key 通过 Chrome 内置存储保存在本地设备，不会发送给任何第三方服务器（除你自己配置的 AI 接口地址外）。扩展仅在执行任务时读取并操作当前网页内容。
