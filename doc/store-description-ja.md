Buddy AI は、自然言語でブラウザのあらゆる操作を自動化できる強力な Chrome 拡張機能です。やりたいことを伝えるだけで、あとはすべて自動でこなします。

できること：
日常的な言葉でタスクを入力するだけで、AI がブラウザを操作してステップごとに完了させます。1回の指示で複数の操作を連続して実行することも可能です。情報検索、フォーム入力、複雑なサイトのナビゲーション、Web アプリの操作など、あらゆる作業を自動で処理します。

自動化できるタスクの例：
- 「GitHub ユーザー sazima のプロフィールを開いて、リポジトリを 1 つスターする」
- 「シンガポールの天気を調べて」
- 「明日の北京〜上海間で最安値のフライトを探して」
- 「Amazon で一番安いメカニカルキーボードを見つけて」
- 「この登録フォームを入力して」
- 「YouTube で GitHub チュートリアルを見つけていいねを押して、シンガポールの天気を調べて、GitHub の sazima/test リポジトリにチュートリアルリンクと現地天気を添えた催促 issue を作成して」

主な機能：
- ✅ 自然言語コントロール — コーディング不要、普通の言葉でタスクを指示
- ✅ マルチステップタスクの連続実行 — 1 文で複雑な一連の操作を完了
- ✅ あらゆるサイトに対応 — サイトごとの設定不要
- ✅ リアルタイムステップ表示 — AI が何をしているか常に確認できる
- ✅ スマートフォーム入力 — 1 回の指示で複数フィールドを一括入力
- ✅ 自動停止保護 — ログイン壁・無限ループ・フリーズを自動検出
- ✅ 自分の API キーを使用 — DeepSeek・OpenAI・OpenAI 互換 API に対応
- ✅ 柔軟な設定 — API Base URL と最大ステップ数をカスタマイズ可能
- ✅ 多言語 UI — English・简体中文・繁體中文・日本語・한국어 に対応

なぜインストールするべきか：
繰り返しのブラウザ操作に時間を費やすのはもうやめましょう。Buddy AI は複雑な多段階の Web 作業を 1 文で実現します。ブラウザを操作できるパーソナルアシスタントを手に入れましょう — データはローカルで処理され、プライバシーはあなたが管理します。

プライバシーについて：
API キーは Chrome の組み込みストレージを使ってお使いのデバイスにローカル保存されます。あなたが設定した AI API エンドポイント以外のサードパーティサーバーにはデータを送信しません。拡張機能はタスク実行時のみ現在のページの内容を読み取り・操作します。
