Buddy AI는 자연어로 모든 웹 작업을 자동화할 수 있는 강력한 Chrome 확장 프로그램입니다. 원하는 것을 말하면 나머지는 알아서 처리합니다.

무엇을 할 수 있나요：
일상적인 언어로 작업을 입력하면 AI가 브라우저를 제어하여 단계별로 완료합니다. 한 번의 명령으로 여러 작업을 순서대로 연속 실행하는 것도 가능합니다. 정보 검색, 양식 작성, 복잡한 사이트 탐색, 웹 앱 조작 등 모든 작업을 자동으로 처리합니다.

자동화할 수 있는 작업 예시：
- "GitHub 사용자 sazima의 프로필을 열고 저장소 하나를 스타하기"
- "싱가포르 날씨 알려줘"
- "내일 베이징에서 상하이 가는 가장 저렴한 항공편 검색해줘"
- "아마존에서 가장 저렴한 기계식 키보드 찾아줘"
- "이 회원가입 양식 작성해줘"
- "YouTube에서 GitHub 튜토리얼 찾아서 좋아요 누르고, 싱가포르 날씨 확인한 다음, GitHub sazima/test 저장소에 튜토리얼 링크와 현지 날씨를 첨부한 업데이트 요청 이슈 만들어줘"

주요 기능：
- ✅ 자연어 제어 — 코딩 없이 일상 언어로 작업 지시
- ✅ 멀티스텝 작업 연속 실행 — 한 문장으로 복잡한 순차 작업 완료
- ✅ 모든 웹사이트 지원 — 사이트별 별도 설정 불필요
- ✅ 실시간 단계 표시 — AI가 무엇을 하고 있는지 항상 확인 가능
- ✅ 스마트 양식 작성 — 한 번의 명령으로 여러 필드 일괄 입력
- ✅ 자동 보호 기능 — 로그인 벽·무한 루프·페이지 멈춤 자동 감지
- ✅ 내 API 키 사용 — DeepSeek·OpenAI·OpenAI 호환 API 지원
- ✅ 유연한 설정 — API Base URL 및 최대 단계 수 커스터마이징 가능
- ✅ 다국어 UI — English·简体中文·繁體中文·日本語·한국어 지원

왜 설치해야 하나요？
반복적인 브라우저 작업에 시간을 낭비하지 마세요. Buddy AI는 복잡한 다단계 웹 워크플로우를 한 문장으로 처리합니다. 브라우저를 조작할 수 있는 개인 비서를 갖는 것과 같습니다 — 데이터는 로컬에서 처리되고 개인 정보는 내가 관리합니다.

개인 정보 보호：
API 키는 Chrome 내장 저장소를 사용하여 기기에 로컬로 저장됩니다. 직접 설정한 AI API 엔드포인트 외에는 어떤 서드파티 서버에도 데이터를 전송하지 않습니다. 확장 프로그램은 작업 실행 시에만 현재 페이지의 내용을 읽고 조작합니다.
