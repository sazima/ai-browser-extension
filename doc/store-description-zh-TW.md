Buddy AI 是一款強大的 Chrome 擴充功能，讓你用自然語言自動完成任何網頁操作——只需告訴它你想做什麼，剩下的交給它搞定。

它能做什麼：
只需用日常語言輸入一個任務，AI 就會接管你的瀏覽器，一步一步地完成它——支援在一次指令中串行完成多個操作。無論是搜尋資訊、填寫表單、瀏覽複雜網站，還是操作 Web 應用，Buddy AI 都能全自動處理。

可以自動化的任務範例：
- "打開 GitHub 用戶 sazima 的主頁，並 Star 他的一個倉庫"
- "幫我查詢新加坡的天氣"
- "幫我搜尋明天北京到上海最便宜的機票"
- "在亞馬遜上找最便宜的機械鍵盤"
- "幫我填寫這個註冊表單"
- "去 YouTube 找一個 GitHub 教學並按讚，然後查新加坡天氣，最後在 GitHub 的 sazima/test 倉庫建立一個催更 issue，附上教學連結和當地天氣"

核心功能：
- ✅ 自然語言控制——用日常語言描述任務，無需任何程式設計
- ✅ 多步驟任務串行——一句話完成複雜的連續操作
- ✅ 支援任意網站——無需針對特定網站做任何設定
- ✅ 即時步驟展示——隨時看到 AI 正在執行哪一步
- ✅ 智慧批次填表——一次指令填寫所有欄位
- ✅ 自動保護機制——自動識別登入牆、無窮迴圈和頁面卡死
- ✅ 使用自己的 API Key——相容 DeepSeek、OpenAI 及任何 OpenAI 格式介面
- ✅ 彈性設定——可自訂 API Base URL 和最大步數
- ✅ 多語言介面——支援 English、简体中文、繁體中文、日本語、한국어

為什麼要安裝它？
不要再把時間浪費在重複的瀏覽器操作上。Buddy AI 讓複雜的多步驟網頁任務變成一句話。就像擁有一個能操控瀏覽器的私人助手——資料本地處理，隱私完全由你掌控。

隱私說明：
你的 API Key 透過 Chrome 內建儲存保存在本地裝置，不會發送給任何第三方伺服器（除你自己設定的 AI 介面地址外）。擴充功能僅在執行任務時讀取並操作目前網頁內容。
