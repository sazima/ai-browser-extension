# AI ブラウザアシスタント

AIがブラウザ操作を自動化するChrome拡張機能。やりたいことを自然言語で伝えるだけで、AIがブラウザを操作して完了させます。

## 機能

- **自然言語で操作** — やりたいことを伝えるだけ、あとはAIにおまかせ
- **あらゆるサイトに対応** — サイトごとの設定不要
- **リアルタイムステップ表示** — AIが何をしているか常に確認できる
- **フォーム一括入力** — 複数フィールドを一度に入力
- **要素のビジュアル表示** — インタラクティブな要素すべてに番号が表示される
- **自動保護機能** — ログイン壁・無限ループ・ページ停止を自動検出
- **自分のAPIキーを使用** — DeepSeek・OpenAI・OpenAI互換APIに対応
- **多言語UI** — English、简体中文、繁體中文、日本語、한국어

## インストール

### Chrome ウェブストアからインストール
*(近日公開予定)*

### 手動インストール（デベロッパーモード）

1. **コードをダウンロード** — このページの緑色の **Code** ボタン → **Download ZIP** をクリックして解凍。またはGitを使用：`git clone <リポジトリURL>`
2. Chromeを開き、アドレスバーに **`chrome://extensions`** と入力してEnter
3. 右上の **デベロッパーモード** をオンにする
4. **パッケージ化されていない拡張機能を読み込む** をクリック
5. 解凍したフォルダ内の **`src`** フォルダを選択（ルートフォルダではなく）
6. 拡張機能アイコンがChromeツールバーに表示されます — クリックしてサイドパネルを開く

## 初期設定

1. 拡張機能アイコンをクリックしてサイドパネルを開く
2. ⚙️ **設定** をクリック
3. APIキーを入力（DeepSeek / OpenAI / その他の互換APIキー）
4. 任意：カスタム **API Base URL** を設定（デフォルト：`https://api.deepseek.com/v1`）
5. **保存** をクリック

## 使用例

任意のウェブページを開き、チャットボックスにタスクを入力：

```
GitHubのsazimaユーザーページを開いて、リポジトリの一つにStarをつけて
シンガポールの天気を調べて
明日の北京から上海への最安値フライトを検索して
このページの登録フォームを入力して
```

AIが各ステップをリアルタイムで表示します。**■** をクリックしていつでも停止できます。

## 設定項目

| 設定 | 説明 | デフォルト |
|------|------|-----------|
| APIキー | OpenAI互換のAPIキー | — |
| API Base URL | APIエンドポイント | `https://api.deepseek.com/v1` |
| 最大ステップ数 | 自動停止までの最大ステップ数 | 60 |
| 言語 | UIとAI返答の言語 | ブラウザの言語 |

## 権限について

| 権限 | 理由 |
|------|------|
| `<all_urls>` | ユーザーが自動化したい任意のサイトのコンテンツを読み取り操作するため |
| `activeTab` | 現在のアクティブなタブにアクセスするため |
| `scripting` | ページ要素を操作するためにコンテンツスクリプトを注入するため |
| `sidePanel` | ChromeのサイドパネルにチャットUIを表示するため |
| `storage` | APIキーと設定をローカルに保存するため |
| `tabs` | アクティブなタブのIDとURLを取得するため |

APIキーはデバイスにローカル保存され、設定したAI APIエンドポイント以外には送信されません。
